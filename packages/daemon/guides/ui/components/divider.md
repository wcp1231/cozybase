# Divider

Horizontal separator line with optional label text.

## Properties

| Property | Type | Required | Description |
|----------|------|----------|-------------|
| `label` | string | No | Text displayed on the divider |

## Examples

```json
{ "type": "divider" }
```

With label:

```json
{ "type": "divider", "label": "Advanced Settings" }
```
